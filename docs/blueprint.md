# **App Name**: GuardianWake

## Core Features:

- Intelligent Alarm Scheduler: Custom interface for setting primary and secondary wake-up times with intuitive time pickers.
- Backup Contact Integration: Secure configuration area to save and verify the emergency contact phone number.
- Non-Response Monitoring: Detection system that starts a 5-minute silent countdown if the primary alarm is not dismissed.
- AI Speech Tool: Generates a customized tool script for the backup call, using AI to synthesize a personalized emergency message for the recipient.
- Cloud Telephony Trigger: External API integration to automatically place the phone call once the 5-minute timeout occurs.
- Awake Verification Dashboard: Visual status indicator showing whether the system is active, in standby, or currently triggering an emergency call.

## Style Guidelines:

- Primary color: Vivid Indigo (#8278f2) representing a sense of reliable transition from night to dawn.
- Background color: Deep Charcoal-Purple (#16141f) for a focused, low-eye-strain nighttime interface.
- Accent color: Electrifying Blue (#2e81ff) used for critical UI elements and active status indicators.
- Headline and body font: 'Space Grotesk', a geometric sans-serif that provides a tech-centric, precise look similar to scientific digital instruments.
- Outlined icons with neon glows to emphasize active alarm states and telephony connections.
- Centralized, oversized clock face design with critical 'dismiss' controls prioritized to avoid accidental misclicks.
- Gentle pulsating ambient animation behind the timer to indicate the 5-minute grace period is ticking down.
